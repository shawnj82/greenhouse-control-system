üìã Configuration Guidelines

Backend Cache TTL:

  • 1-2 seconds: Real-time updates, high sensor load
  • 5 seconds: Default, good balance of responsiveness and efficiency
  • 10-15 seconds: Reduced sensor load, still responsive
  • 30+ seconds: Minimal sensor load, slower updates

Frontend Update Interval:

  • 1-5 seconds: Real-time monitoring, high network usage
  • 10 seconds: Default, responsive for active monitoring
  • 30-60 seconds: Background monitoring, minimal network usage
  • 120+ seconds: Periodic checks, very low network usage

Performance Considerations:

  • TCS34725 sensor readings take ~500ms due to 240ms integration time
  • Light conditions in greenhouses change slowly (minutes, not seconds)
